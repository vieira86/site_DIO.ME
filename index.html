Entenda as Principais Tags HTML

<html>

Define o início e o fim de um documento HTML. É o elemento raiz de qualquer página HTML.

<head>

Contém informações sobre o documento, como metadados, título, links para estilos e scripts.

<body>

Contém todo o conteúdo visível da página, como textos, imagens, links, tabelas etc.

<title>

Define o título da página exibido na aba do navegador.

<meta>

Fornece metadados como codificação de caracteres e configurações de visualização responsiva.

<h1>

Define um título de nível 1. Existem também `h2` até `h6` para subtítulos hierárquicos.

<p>

Define um parágrafo de texto.

<a>

Cria um hiperlink para outra página ou recurso.
